Q : 有时候怎么一直都获取不到图片，显示于服务器通讯失败啊？我这里网络一切正常啊。 A : 你可以试试访问网页版来看看是否能打开网页。
  • 如果不能打开网页，说明服务器发生了错误导致网站挂了，请点击反馈反馈一下问题哈。
  • 如果网页能访问，但是网页上显示没有图片，页码也显示不正常，则说明服务器数据库连接断开了，还请点击反馈反馈下问题哈。
  • 如果网页打开正常，这个我也不知道是哪个地方出了问题，也还是请点击反馈反馈下问题哈。
Q : 为什么有时候下载到的图片不对啊？ A : 如果你下载到的是这样的图片的话 亦或是这样的图片 那时因为微软他们也提供这样的图片，这个我也没办法。
Q : 为什么这里看到的和我自己在必应上看到的图片是不一样的？为什么标题是日文的？ A : 因为服务器在日本，所以获取到的也就是日本Bing里面的图片啦，所以你看到的标题等等的都是日文。
Q : 我下载到的图片放在那里了呢？ A : 在 根目录/BingImage/download这个下面
